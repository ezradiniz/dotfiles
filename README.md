# dotfiles

My personal dotfiles

### How to install?

```console
$ git clone --depth=1 https://github.com/ezradiniz/dotfiles.git
$ cd dotfiles
$ make
```

### Using the Makefile

```console
$ make nvim 				# Update nvim
$ make zsh 				# Update zsh
$ make tmux 				# Update tmux
$ make i3 				# Update i3
$ make alacritty 			# Update alacritty
$ make git 				# Update git
$ make x11 				# Update X11
```
